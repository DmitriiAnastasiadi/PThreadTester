# Рабочая тетрадь 3. Задание 3

Напишите программу, которая использует POSIX Threads (Pthreads) для выполнения параллельных вычислений. Программа должна создавать несколько потоков, каждый из которых выполняет свою задачу (например, вычисление суммы элементов массива). После завершения работы всех потоков программа должна вывести результаты на экран. Программа должна быть адаптирована для работы в MINIX 3.

Требования:
1. Программа должна использовать POSIX-функции для работы с потоками (pthread_create, pthread_join, pthread_exit).
2. Программа должна быть адаптирована для MINIX 3, если требуется замена специфичных функций.
3. Программа должна быть скомпилирована и протестирована в MINIX 3.
4. Убедитесь, что программа корректно работает с памятью и не содержит утечек.
